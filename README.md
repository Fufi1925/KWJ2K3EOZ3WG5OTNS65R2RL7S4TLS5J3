# Fufihub

Gaming-Website mit schwarzem Glass-Design:
- Login mit Benutzername + Passwort
- 120 interne spielbare Games (ohne externe Weiterleitung)
- Verbesserte Game-Ansicht für alle Nutzer

## Start
```bash
npm start
```

Danach im Browser öffnen:
`http://localhost:3000`
