# Fufihub

Gaming-Website mit schwarzem Glass-Design:
- Login mit Benutzername + Passwort
- 50 neue Spiele-Ideen (Action/Adventure Fokus)
- Klick auf "Jetzt spielen" öffnet eine eigene Spielseite mit Zurück-Button

## Start
```bash
npm start
```

Danach im Browser öffnen:
`http://localhost:3000`
